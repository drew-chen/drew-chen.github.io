Features

Based on Git distributed version control.
Supports the follow git commands:

  • init: Initialize the repository in the current directory
  • add [file name]: Stage changes of a file
  • commit [message]: Save a snapshot of committed and staged files
  • rm [file name]: Unstage and delete the file if a file staged or committed
  • log: Display information about each commit starting from the head to the initial commit
  • global-log: Display information about every commit ever made
  • find [commit message]: Print out the commit(s) that have the given message
  • status: Display information about all branches, the stage, and file modifications
  • checkout -- [file name]: Overwrites the directory with the version of the file as it exists in the head commit
  • checkout [commit id] -- [file name]: Overwrites the directory with the version of the file as it exists in the given commit
  • checkout [branch name]: Overwrites current directory with all files tracked in the given branch
  • branch [branch name]: Creates a new branch with the given name
  • rm-branch [branch name]: Removes the branch with the given name
  • reset [commit id]: Checks out all files tracked by the given commit
  • merge [branch name]: Merges files from the given branch into the current branch
